== Python

Coisas sobre python

=== Instalando o pip

----
sudo apt install python3-pip
----

Instalando o pipenv:

----
pip3 install pipenv
----

=== Criando ambiente virtual



==== Com virtualvenv

----
virtualenv venv --python `which python3`
----

==== Com pipenv

No Ubuntu:

----
sudo apt install python3-pip
pip3 install pipenv
----
